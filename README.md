## Minimal, Vanilla TypeScript Website üç¶

To use: 
### Via clone:
1. clone the repository
2. `$ cd template-minimal-vanilla-ts && rm -rf .git` to remove .git for this template repo


### Via 'Use this template':
1. Click the 'Use this template' then `pull` it local


### On your machine:
1. `$ npm install`
2. inside of [`webpack.config.js`](https://github.com/MisterPea/template-minimal-vanilla-ts/blob/65b497cbb63b20435f7486d1c12ca720970a26d6/webpack.config.js#L42) change the `host:192.168.1.xxx` to your i.p.
3. make cool stuff!
